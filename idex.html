💡 Kesimpulan Pembelajaran

Melalui kelas Belajar Dasar Pemrograman Web di Dicoding ini, saya berhasil memahami fondasi utama front-end development, mulai dari penyusunan struktur menggunakan Semantic HTML, dekorasi layout responsif murni dengan CSS Flexbox, hingga penerapan prinsip pemisahan kode (*separation of concerns*).

🚀 Harapan & Impact Bagi Saya

Sebagai mahasiswa Teknologi Informasi yang memiliki minat di bidang IT, kelas ini memberikan dampak yang luar biasa untuk saya, mengenai baimana cara mengatur responsive website,dan dasar dasar pembuatan web,melalui tat letak elemen halaman web dengan css flexbox. Saya kini mampu membuat website biodata diri dengan statis saya sendiri untuk mendemonstrasikan aset ilustrasi digital saya tanpa ketergantungan pada platform pihak ketiga. Harapan saya ke depan adalah bisa menempertajam keahlian full-stack development dengan skill yang saya dapatkan dari kelas ini untuk menciptakan produk web yang lebih interaktif,modern dan fungsional.

🙏 Tanda Terima Kasih
